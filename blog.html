Discuss the scope of var, let and const.

var: Variables declared with var can be redeclared and reassigned, as well as accessed from wherever it is declared.

let: Variables declared with let can be reassigned and redeclared only within the block scope in which they are declared.

const: Variables declared with const can only be redeclared within the block scope in which they are declared but not reassigned.

Tell us the use cases of null and undefined.

undefined:

Default Value: If something is declared in a variable without being assigned.

Function Parameters: When a function is called without providing a value to the function's parameter.

Missing Object Properties: Attempting to access a non-existent property on an object.

Return Value: If a function does not return a value specified.

Also can be undefined for some other reasons.

null:

Assigning to Indicate No Value: Assigning 'null' to a variable or property value is specified to be empty.

Also can be null for some other reasons.

What do you mean by REST API?

REST API is basically Representational State Transfer Application Programming Interface. It is a style of designing network based applications. It helps in fast communication between web server and client. It uses standard HTTP methods. Some common methods are:

Get: Used to receive any data from the server.

Post: Used to provide any data to the server.

Put: Used to update any data on the server.

Delete: Used to delete any data from the server.
